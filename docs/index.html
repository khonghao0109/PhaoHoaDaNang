File được commit thẳng vào repo — không cần token ở trình duyệt
📖 Hướng dẫn setup Worker lần đầu (click để mở)
1
Tạo Fine-grained PAT trên GitHub
Vào github.com/settings/tokens → New fine-grained token → chọn repo → tick Contents: Read & Write → Generate.
2
Deploy Cloudflare Worker
Copy src/worker.js vào project, chạy: 
wrangler deploy
3
Đặt secrets
wrangler secret put GITHUB_TOKEN
wrangler secret put GITHUB_OWNER
wrangler secret put GITHUB_REPO
wrangler secret put GITHUB_BRANCH
4
Nhấn ✏️ để đổi Worker URL nếu cần
URL mặc định đã được cấu hình sẵn trong code — chỉ cần mở tab GitHub, chọn folder và kéo thả file.
